# Simtools-Projekt

## Voraussetzungen
Um einen reibungslosen optischen Ablauf des Programms zu gewährleisten, wird eine Bildschirmauflösung von mindestens 1680 x 1050 Pixel empfohlen.

### 1. Virtuelle Umgebung erstellen
Führen Sie folgenden Befehl im Projektordner aus:
```bash
python3 -m venv venv
```

### 2. Virtuelle Umgebung aktivieren
**Mac/Linux:**
```bash
source venv/bin/activate
```
**Windows (PowerShell):**
```powershell
venv\Scripts\Activate.ps1
```

### 3. Abhängigkeiten installieren
Python Version 3.11 ist erforderlich, höhere Versionen werden nicht zwingend unterstützt. Installieren Sie die benötigten Bibliotheken mit:
```bash
pip install -r requirements.txt
```

### 4. Anwendung starten
Starten Sie das Programm mit:
```bash
python main.py
```
